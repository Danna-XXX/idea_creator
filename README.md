# 灵感铺子

> AI 短视频内容创作助手 · 专为小红书/抖音个人博主设计

帮博主解决三件事：**起号定位** · **选题策划** · **脚本生成**

不是帮你写内容，而是拆解爆款背后的逻辑，让你用起来心里有底。

---

## 技术栈

- **框架**：React 18 + Vite 5
- **样式**：Tailwind CSS v3（移动端优先，390px 宽度，暗色主题）
- **路由**：React Router v6
- **AI**：[硅基流动](https://siliconflow.cn) SiliconFlow API（OpenAI 兼容，DeepSeek-V3 模型）
- **导出**：SheetJS (xlsx)
- **数据持久化**：localStorage（用户人设、灵感记录、自定义公式、头像、账号信息、各账号独立的免费次数）

## 快速开始

### 前置要求

- Node.js ≥ 18
- npm ≥ 9

### 安装

```bash
git clone <your-repo-url>
cd DXMAI/demo
npm install
```

### 环境变量配置

在 `demo/` 目录下新建 `.env.local` 文件：

```
VITE_SILICONFLOW_API_KEY=sk-xxxxxxxxxxxxxxxx
```

**⚠️ 注意：`=` 后面不能有空格**，否则 API Key 会带空格导致认证失败。

**获取 API Key：**
1. 前往 [硅基流动官网](https://siliconflow.cn) 注册账号
2. 进入控制台 → API 密钥 → 新建密钥
3. 复制后粘贴到 `.env.local`，注册即送免费额度

> **没有 API Key？** 也可以直接体验——系统会自动使用内置 Mock 数据，所有 UI 功能均可正常浏览，脚本生成会返回示例数据。

### 启动开发服务器

```bash
cd demo
npm run dev
```

打开 [http://localhost:3000](http://localhost:3000)（端口被占用时自动切换到 3001）。

在浏览器开发者工具中将设备设置为 **iPhone 15 Pro（390×844）** 可获得最佳体验。

## 使用流程

**推荐体验路径：**

1. **首次使用** → 点击右上角头像 → 设置昵称和头像
2. **建立人设** → 账号定位向导（3步，约1分钟）
3. **找选题** → 选题库会自动高亮匹配你赛道的选题 → 点击"生成脚本"
4. **看公式** → 公式库 → 选一个感兴趣的公式 → "用此公式"
5. **生成脚本** → 填写选题 → 选时长/风格 → 点击生成 → 可继续润色
6. **下载** → 右上角"下载 Excel"→ 得到可直接用于拍摄的分镜表格

**视频链接分析：**

在公式库或脚本生成器中粘贴小红书/抖音/B站视频链接，AI 会根据平台风格自动生成参考公式或推断选题。

> 注：受平台爬虫限制，无法读取视频实际内容，AI 根据平台特征生成参考结构。

## 常见问题

**Q：运行后看到的是 Mock 数据还是 AI 数据？**
A：取决于 `.env.local` 是否配置了有效的 API Key。有 Key → AI 实时生成；无 Key → 自动 fallback 到内置 Mock 数据，不影响 UI 体验。

**Q：端口 3000 被占用怎么办？**
A：Vite 会自动切换到 3001，终端会显示实际地址，直接访问即可。

**Q：头像上传后刷新会丢失吗？**
A：不会，头像以 base64 格式存入 localStorage，刷新和重启浏览器后均保留。

**Q：公式库粘贴视频链接为什么不能显示视频内容？**
A：读取平台视频内容需要爬虫/API，属于后端技术，Demo 阶段暂不实现。当前方案：AI 根据链接的平台（小红书/抖音/B站）生成该平台风格的参考公式。

**Q：登录账号有什么好处？**
A：未登录状态下，免费次数（数据复盘3次 / Excel下载5次）按设备共用，同一设备所有人共享计数。登录后每个账号拥有独立的免费次数，互不影响。账号信息存储在本地设备，无需联网验证。
