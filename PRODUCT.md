## 机会判断

### 市场背景

短视频已成为个人 IP 成长和商家获客的核心渠道。小红书月活超 3 亿，抖音日活超 7 亿，但绝大多数创作者（>90%）粉丝量停留在 1 万以下。这批人数量庞大、付费意愿强、但缺乏专业的内容运营能力。

### 目标用户

**小红书 / 抖音上粉丝量 1 万以下的个人创作者**，尤其是：
- 已经开始更新但增粉缓慢、找不到方向的新博主
- 想把账号做起来但不知道从哪里入手的素人
- 有内容想法但不知道怎么包装和表达的创作者

### 核心痛点

| 痛点 | 具体表现 |
|------|---------|
| **起号难** | 账号定位模糊，发了10条内容风格各异，算法无法识别账号标签，流量池一直很小 |
| **选题难** | 不知道今天拍什么，刷到爆款也不知道为什么爆，跟风拍效果也差 |
| **话术难** | 标题写不出吸引力，开头三秒留不住人，脚本逻辑混乱，完播率低 |

### 为什么现在是时机

1. **内容策略端存在空白**：现有 AI 工具（剪映、一帧秒创）主攻剪辑和视频生成，没有产品专门解决"拍什么、怎么说"的策略问题
2. **AI 能力刚好匹配**：LLM 可以拆解视频结构、提炼语言规律、生成对标内容，技术可行性已具备
3. **创作者付费意愿已验证**：小红书/抖音评论区大量博主付费购买"爆款选题""脚本模板"类课程，说明需求真实存在
4. **格局未定**：垂直于"内容策略"的 AI 工具赛道尚无强势玩家

## 产品介绍

### 核心差异：公式拆解 + 对比生成

市面上的 AI 写作工具都是"黑盒生成"——你输入主题，AI 输出内容，用户不知道为什么这样写、能不能信任这个结果。

灵感铺子的逻辑是**"先让你看懂爆款，再帮你复用逻辑"**：
1. AI 拆解真实爆款视频的结构、节奏和话术意图
2. 提供可验证的模板（你能看到原视频的数据）
3. 生成内容时与原版公式并排对比，用户自己判断质量

信任来自透明，不来自承诺。

### 功能架构（V1 个人博主专版）

#### 1. 账号定位向导
3步完成人设搭建：
- 选择内容赛道（美食/养生/影视解说/游戏/穿搭等 15 个垂直方向）
- 选择内容风格（温暖真实 / 专业干货 / 小众品味）
- 生成人设标签 + 推荐选题方向 + 风格关键词

人设建立后，选题库会自动按赛道优先排序。

#### 2. 选题库
- 按内容类型分类（Vlog、测评、知识、美食、养生、影视等 16 个类别）
- 每个选题标注：**爆款潜力评级**（⭐×1-5）+ **近期热度趋势**（上升/稳定）+ **难度等级**
- 30+ 精选选题，持续扩充

#### 3. 爆款公式库（核心差异点）
两种来源：
- **平台精选**：7 条精选公式，标注真实数据（点赞、收藏、评论数）
- **用户导入**：粘贴视频链接（小红书/抖音/B站），AI 根据平台风格自动生成公式结构

每条公式展示：
- 视频来源和真实数据
- 逐段拆解（开头 / 中段 / 结尾）
- 每段的**设计意图**注释（为什么这样说，达到什么效果）
- **新鲜度指标**（近期使用量，避免跟风撞款）

#### 4. 脚本生成器（AI 驱动）
- 时长选择：15s / 30s / 60s / 90s / 3分钟 / 5分钟 / 自定义
- 内容类型 × 风格双维度配置
- 支持绑定公式：AI 严格按照所选公式的段落节奏生成脚本
- 视频链接灵感导入：粘贴链接，AI 自动推断选题和内容类型
- **脚本润色**：生成后可告诉 AI 如何修改，支持多轮对话调整
- 生成结果包含：标题建议、钩子话术、分镜表（含 BGM 真实歌曲推荐）、封面文案、话题标签、发布时间建议
- 一键导出 Excel 分镜表格

#### 5. 随手记 → 灵感库
- 看到有意思的内容、突然有想法，一键记录
- 浏览历史灵感，支持删除

**付费功能（规划中）：**
- AI 定期扫描灵感库，自动推荐相关选题
- 情绪陪伴回应

#### 6. 个人主页
- 头像上传（本地存储）
- 昵称设置
- 功能导航入口（数据复盘、我的人设、我的公式等）

#### 7. 效果追踪与复盘（付费功能）
- 视频播放量/完播率趋势图
- 爆款因素归因分析
- AI 个性化优化建议
- 账号成长周报

### 核心用户流程

```
账号定位 → 选题 → 套用公式 → 生成脚本 → 下载 Excel → 发布 → 数据复盘（付费）
    ↑                                                               ↓
    └──────────────── 随手记随时补充灵感 ←──────────────────────────┘
```

## 商业判断

### 用户为什么付费

| 价值 | 具体体现 |
|------|---------|
| **省时间** | 每条视频内容策划从 2 小时压缩到 20 分钟 |
| **提胜率** | 有真实数据验证的公式，比自己瞎摸成功率高 |
| **替代专业运营** | 请一个内容运营每月需要 5000 元以上，39 元订阅替代核心价值 |
| **成长可见** | 效果复盘让博主看到自己在进步，有持续使用动力 |

### 定价方案

| 版本 | 价格 | 包含功能 |
|------|------|---------|
| **免费版** | 0元 | 每月5次AI脚本生成、基础选题库（20个）、公式库浏览、随手记 |
| **Pro 版** | 39元/月 | 无限AI脚本生成、完整选题库（30+）、完整公式库 + 搜索、AI 脚本润色、Excel 下载 |
| **高级版** | 99元/月 | Pro 全部功能 + AI 灵感分析、个性化选题推荐、数据复盘报告、AI 情绪陪伴 |

**定价逻辑：**
- 39元 = 一杯咖啡，解决每天"今天拍什么"的焦虑，门槛低、易转化
- 99元 = 替代部分内容运营职能，面向认真把账号当事业的博主
- 年付打85折，提升 LTV

### 获客策略

**冷启动阶段（0-1000用户）：**
1. **产品即渠道**：用灵感铺子做内容，在小红书/抖音发"AI 帮我30天涨粉X"的真实记录，产品曝光和内容价值二合一
2. **社群渗透**：进入博主/创作者微信群，提供免费的"选题诊断"服务，建立口碑
3. **KOC 测评**：邀请 1000-5000 粉的小博主免费使用，换取真实测评内容

**增长阶段（1000+用户）：**
4. **免费工具钩子**：上线独立的"标题吸引力检测器""7天选题生成器"等免费小工具，作为流量入口
5. **KOL 合作**：与内容创作类博主（教博主涨粉的账号）合作测评
6. **老带新裂变**：推荐1名好友注册，双方各得7天高级版体验

### 关键商业指标

- **目标转化率**：免费→Pro 转化 ≥ 15%
- **月留存率**：付费用户月留存 ≥ 70%
- **回收周期**：单用户获客成本（CAC）< 3个月订阅费

## 2周验证计划

验证目标：在不写一行代码的前提下，验证"用户是否真的存在这个痛点"和"用户是否愿意为解决方案付费"。

### 第一周 — 用户与问题验证

**Day 1-2：招募目标用户**
- 目标：找到 20 名 <1 万粉的博主（小红书或抖音）
- 渠道：创作者微信群、小红书私信（搜索"起号"相关笔记）、豆瓣博主小组
- 筛选标准：已经在更新内容但增长停滞，或想开始做但不知道怎么起步

**Day 3-4：用户访谈（10 场）**
- 每场 30 分钟，核心问题：
  - 你现在每周花多少时间在内容策划上？
  - 最让你头疼的环节是什么？（选题 / 标题 / 脚本 / 发布后没流量）
  - 你现在用什么方法解决这个问题？（找课程 / 抄爆款 / 随机发）
  - 如果有工具帮你做这件事，你愿意每月花多少钱？
- 重点验证：痛点优先级是否和假设一致，付费意愿和价格敏感点在哪里

**Day 5-7：制作核心流程原型**
- 工具：Figma（免费）或直接用飞书文档模拟
- 覆盖主流程：账号定位 → 选题 → 公式参考 → 脚本生成
- 不需要写代码，纯静态原型即可

### 第二周 — 方案验证

**Day 8-9：手工搭建最小版公式库**
- 手动整理 20-30 个小红书/抖音真实爆款视频
- 人工拆解每条的结构、话术和设计意图（不用 AI，验证内容质量）
- 评估：用户看完这些拆解，能不能直接上手用？

**Day 10-11：原型用户测试（10 人）**
- 让用户使用原型完成"从选题到生成第一条视频脚本"的完整流程
- 观察指标：
  - 任务完成率（不被卡住的比例）
  - "啊哈时刻"（用户说"这个好！"的瞬间是哪里）
  - 自发行为（用户有没有主动截图、发给朋友看）

**Day 12-13：搭建落地页 + 测试付费意愿**
- 工具：Notion / Carrd / 飞书（快速搭建，无需开发）
- 页面包含：产品价值主张 + 功能介绍 + 定价方案 + "开始免费体验"按钮
- 监测：页面访问量、按钮点击率、愿意留邮箱的比例

**Day 14：汇总数据，Go / No-Go 决策**

### 验证成功标准

| 指标 | 成功门槛 |
|------|---------|
| 付费意愿 | 40%+ 受访用户表示愿意为 Pro 版（39元/月）付费 |
| 流程可用性 | 70%+ 用户不需要引导就能独立完成核心流程 |
| 自发传播 | 至少 3 名测试用户主动推荐给其他博主 |
| 落地页转化 | 访问落地页的用户中 20%+ 留下联系方式 |

任意 3 项达标 → 进入 MVP 开发阶段。

### 资源投入

| 资源 | 投入 |
|------|------|
| 人力 | 1-2 人 × 2 周（产品经理为主，无需工程师） |
| 工具 | Figma（免费）、飞书文档（免费）、Notion/Carrd（免费） |
| 资金 | 接近 0，主要是时间成本 |
| 风险 | 最大损失 = 2 周时间，不存在技术债务 |
